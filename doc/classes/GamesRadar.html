setup () {|GamesRadar::Config| ...}

Use this shortcut for GamesRadar::Config. Refer to the GamesRadar::Config for all available options.

Instead of setting values like 

GamesRadar::Config.api_key = "SxKuVse22qqUcZXq"

You can write it as 

GamesRadar.setup do |c|
  c.api_key = "SxKuVse22qqUcZXq"
end
